# Canny Edge Detection

This project is a implementation of the Canny Edge Detection algorithm in C++ and OpenCL. The project is part of the course " Laboratory Course High Performance Programming With Graphic Cards " at the Universit√§t Stuttgart.


## Installation

```bash
git clone https://github.com/DanYuDE/CannyEdgeDetection-GPU.git
```

## Usage

Create a build directory.
```bash
cd CannyEdgeDetection-GPU
mkdir build
cd build
```

Compile the project with meson and ninja.
```bash
meson configure
ninja
```
